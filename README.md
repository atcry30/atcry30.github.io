# Lunvarre | Timeless Precision

Front-end only luxury watch store built with React + Tailwind CSS. All data is client-side via a mock JSON array.

## Features
- Black, gold, white theme with luxurious typography
- Navbar with cart badge (state-based)
- Hero with background image and animated fade-in
- Product collection grid (6â€“8 watches)
- Add to cart, remove, quantity, total, localStorage persistence
- About section and footer with links + social icons
- Smooth scrolling and subtle hover transitions

## Tech
- React 18 + Vite
- Tailwind CSS 3

## Getting Started
```bash
# from the project root
cd lunvarre
npm install
npm run dev
```
Open the shown localhost URL.

## Customization
- Update images and products in `src/data/watches.js`
- Theme in `tailwind.config.js` (colors, fonts)
- Hero image in `src/components/Hero.jsx`

## Notes
- The cart is demo-only (no checkout backend).
- Images reference public URLs (requested: Pinterest/online). Replace with your own assets if needed.
